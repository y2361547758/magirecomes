Magireco MES（α）

Magireco MES (Magireco Mental Enhance Simulator)は、
マギアレコードのキャラ固有能力育成機能「精神強化」のシミュレータです。
ゲーム内では解放が一度きりの精神強化を自由に試すことができます。

【使い方】

「CharacterSelect」より対象キャラを選択すると、
そのキャラの精神強化ボードが生成されます。
(※灰色のキャラは未実装です。)

能力解放はSTARTのマスから次のマスをクリックしていくだけです。

強化後のステータス、及び、現在の能力解放数はボード左上に表示されます。
習得済みのディスク強化・潜在アビリティ・潜在スキルは
ボード右上の「Ability / Skill」をクリックすると一覧が表示されます。

入力状況のセーブ・ロード機能はありません（今後実装予定）。
別のキャラを選択した場合、現在入力しているキャラの状態はリセットされます。
また、同じキャラを再度選択した場合もリセットとなります。

【動作推奨環境】

PC上のChromeでの利用を推奨します。他のブラウザでは確認していません。
マップ右端の列が見えるようウィンドウサイズを調整、またはズーム倍率を縮小してご利用ください。

スマホ・タブレットは非推奨です。
iPadのSafariで確認しましたが、一部機能しないボタンがあります。
スマホではページの横幅が非常に広い影響でほとんど表示しきれません。

【能力解放数(Release)の上限について】

本家同様、デフォルトでは60までを上限としています。
上限数に達している場合は、未取得のマスをクリックしても解放されません。

「cap 60」のチェックを外すと100まで上限開放されます。

【能力解放について】

各マスをクリックすると、そのマスの能力を獲得できます。

ただし、本家同様、解放は「次のマス」しか解放できません。
先のマスをクリックしても一括解放はできません（今後実装予定）。

また、解放済みマスの解除(払い戻し)はできません（今後実装予定）。
再度「CharacterSelect」から同じキャラを選択し、最初からやり直してください。

【素材・消費CCについて】

素材・消費CCの情報表示は未実装です（今後実装予定）。

【キャラの基本ステータスについて】

そのキャラの最大覚醒＆最大LV時のステータスを基本にしています。
メモリア・覚醒素材のセットボーナスは含んでいません。

【NodeSearch(マス検索機能)について】

本家には無い実験的な機能です。
キャラ選択後に「CharacterSelect」の右側に出現します。

本家では獲得したい能力のマスを全体から一つ一つ目で探すのが大変なので、
マス探索の補助を目的として実装しています。
（後々、素材名を基に検索する機能も検討中。）

「NodeSearch」より検索対象を選択すると、対象のマスが波紋表示されます。

検索対象は以下の通り。

None 全マスの波紋表示を停止します。
HP 「HP最大能力値アップ」を持つマスを全て波紋表示します。
HP MAX Node 「HP最大能力値アップ」を持つマスの内、最も高い上昇値を持つマスを波紋表示します。
HP MAX Route 各終点までのルートの内、「HP最大能力値アップ」の上昇値合計が最も高くなるルートを波紋表示します。
ATK 「ATK最大能力値アップ」を持つマスを全て波紋表示します。
ATK MAX Node 「ATK最大能力値アップ」を持つマスの内、最も高い上昇値を持つマスを波紋表示します。
ATK MAX Route 各終点までのルートの内、「ATK最大能力値アップ」の上昇値合計が最も高くなるルートを波紋表示します。
DEF 「DEF最大能力値アップ」を持つマスを全て波紋表示します。
DEF MAX Node 「DEF最大能力値アップ」を持つマスの内、最も高い上昇値を持つマスを波紋表示します。
DEF MAX Route 各終点までのルートの内、「DEF最大能力値アップ」の上昇値合計が最も高くなるルートを波紋表示します。
Ability 「潜在アビリティ」を持つマスを波紋表示します。
Skill 「潜在スキル」を持つマスを波紋表示します。
Disc Enhance 「Accele性能アップ」「Charge性能アップ」「Blast性能アップ」を持つマスを波紋表示します。
Last Node 各ルートにおける終点のマスを波紋表示します。
